## Project info

This is a Vite + React starter tailored for the MaachBazar frontend.

## How can I edit this code?

There are several ways to edit and run the application locally or in the cloud.

### Use the project web UI (if provided)

If this repository is connected to a project dashboard or CI system, use that dashboard to edit or deploy the project. Otherwise, use your local development workflow below.

### Use your preferred IDE

Work locally using your IDE: clone the repo, install deps, and run the dev server.

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm i
npm run dev
```

### Edit a file directly in GitHub

- Navigate to the desired file(s).
- Click the "Edit" button (pencil icon) at the top right of the file view.
- Make your changes and commit them.

### Use GitHub Codespaces

- Click the green "Code" button on the repository page and choose "Codespaces".
- Create a new codespace and edit files directly in the browser-backed IDE.
